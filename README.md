# MatchProphet 🔮

AI-powered World Cup 2026 prediction entertainment. Pick a match, enter your score, get roasted by AI, generate a shareable prophecy card.

**Live:** https://jimmyxu07.github.io/matchprophet/

## Features

- ⚽ Match selector for World Cup 2026 group stage
- 🧠 AI-generated roast + prediction (Gemini 3.5 Flash)
- 📱 Shareable 9:16 prophecy cards (Canvas generation)
- 💼 One-click download + copy text for social media
- 🌐 Pure client-side, zero backend, zero data collection

## Tech Stack

- HTML5 + Tailwind CSS (CDN)
- Vanilla JavaScript
- Google Gemini API (client-side)
- GitHub Pages
